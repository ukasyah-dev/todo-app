# todo-app

A simple todo app made with SvelteKit + Axum.

## Quickstart

Let's start by running the backend. Here's how:

```bash
cd backend
cp .env.example .env
cargo run
```

Open a new terminal, then run these commands:

```bash
cd frontend
pnpm install
pnpm dev
```

Open http://localhost:5173 on your browser.
